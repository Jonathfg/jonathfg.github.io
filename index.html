BEGINNER COURSE

Angular

Aprende a dominar Angular desde los fundamentos hasta conceptos avanzados, con un toque retro.

1

Justificación del Framework

¿Para qué sirve? Angular sirve para desarrollar aplicaciones web modernas, dinámicas y escalables, ofreciendo una arquitectura sólida y componentes reutilizables.

¿Qué aplicaciones tiene? Desde paneles internos de administración, aplicaciones de una sola página (SPAs), e-commerce, hasta aplicaciones móviles híbridas con Ionic.

¿Sobre qué lenguajes se apoya? Principalmente en TypeScript, además de HTML y CSS para la vista y el estilo.

¿Qué prerrequisitos necesita? Conocimientos básicos de HTML, CSS, JavaScript, disposición para aprender TypeScript, y tener instalado Node.js y npm.

2

Instalación

¿Se precisa instalación? Sí, necesitas Node.js, npm y Angular CLI.

Pasos para la instalación: npm install -g @angular/cli, verifica con ng version.

Versiones necesarias: Usa la versión LTS de Node.js y la última versión estable de Angular.

Rutas a tener en cuenta: src/app para componentes, src/assets para estáticos, src/environments para configuraciones.

Variables de entorno: No obligatorias, pero puedes usarlas para endpoints u otras configuraciones.

¿Necesita base de datos? No directamente, Angular consume APIs externas.

3

Primeros pasos

“Hola Mundo”: Crea un proyecto con ng new mi-primera-app, ejecuta ng serve --open y edita app.component.html para mostrar “Hola Mundo desde Angular!”

4

Utilización del Framework

¿Cómo crear una aplicación base? Usa componentes generados con CLI, define rutas, servicios y vistas.

¿Cómo visualizar resultados? Con ng serve en http://localhost:4200, con recarga en vivo. Para producción, ng build --configuration production.

5

Explicación del Funcionamiento

¿Actúa sobre páginas, ficheros, secciones…? Angular actúa sobre componentes (HTML, TS, CSS), organizados en módulos. El enrutador simula múltiples páginas sin recarga completa.

¿Cómo se editan los ficheros? Cada componente tiene su TS, HTML y CSS. Usa data binding y directivas para lógica y vistas dinámicas.

¿Cómo se referencian ficheros desde otras apps? Normalmente se despliega la app Angular como estáticos. Otras apps pueden consumir el frontend final, o integrarlo vía proxies.

6

Conclusiones

Se han cubierto los fundamentos: justificación, instalación, primeros pasos, uso general, funcionamiento interno y conclusiones. Ahora tienes las bases para abordar temas más avanzados de Angular.

7

Enrutamiento

Aprende a configurar rutas, loadChildren para lazy loading, guards y navegación entre componentes.

8

Consumo de APIs

Usa HttpClient para peticiones REST, interceptors para token, manejo de errores y mostrar datos dinámicos.

9

Módulos y Lazy Loading

Organiza tu app en módulos. Aplica lazy loading para cargar módulos a demanda y optimizar el rendimiento inicial.

10

Directivas y Pipes

Crea directivas personalizadas para manipular el DOM y pipes para transformar datos en las vistas.

11

Pruebas (Testing)

Escribe pruebas unitarias con Jasmine y Karma, y pruebas E2E con Protractor, asegurando calidad y fiabilidad.

12

Despliegue

Construye la app para producción con ng build y sube los archivos a un hosting estático o CDN.

13

Seguridad y Autenticación

Implementa autenticación, JWT, guards de ruta y medidas contra XSS e inyecciones.

14

SEO y Metadatos

Usa Angular Universal para SSR, añade metadatos, Open Graph, microdatos y mejora el SEO de tu app.

15

Internacionalización

Traduce tu app a varios idiomas con @angular/localize, archivos XLF, y compila distintas builds por local.

16

Rendimiento y Optimización

Mejora el rendimiento con OnPush, lazy loading, preloading, y analiza la app con Lighthouse.

17

Accesibilidad

Asegura que tu app cumpla estándares de accesibilidad (ARIA, roles, contraste, navegabilidad con teclado).

18

Gestión de Estado

Usa NgRx u otras librerías para un estado predecible, siguiendo patrones de Redux y simplificando la lógica compleja.

19

Animaciones

Agrega animaciones con @angular/animations para transiciones suaves entre vistas y elementos.

20

Herramientas del Ecosistema

Aprovecha Angular CLI, Angular Console, Angular Language Service, y librerías de componentes para acelerar el desarrollo.

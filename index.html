BEGINNER COURSE

Angular

Aprende a dominar Angular desde los fundamentos hasta conceptos avanzados, con un toque retro.

1

Justificación del Framework

¿Para qué sirve? Angular sirve para desarrollar aplicaciones web modernas, dinámicas y escalables, ofreciendo una arquitectura sólida y componentes reutilizables.

¿Qué aplicaciones tiene? Desde paneles internos de administración, aplicaciones de una sola página (SPAs), e-commerce, hasta aplicaciones móviles híbridas con Ionic.

¿Sobre qué lenguajes se apoya? Principalmente en TypeScript, además de HTML y CSS para la vista y el estilo.

¿Qué prerrequisitos necesita? Conocimientos básicos de HTML, CSS, JavaScript, disposición para aprender TypeScript, y tener instalado Node.js y npm.

2

Instalación

¿Se precisa instalación? Sí, necesitas Node.js, npm y Angular CLI.

Pasos para la instalación: npm install -g @angular/cli, verifica con ng version.

Versiones necesarias: Usa la versión LTS de Node.js y la última versión estable de Angular.

Rutas a tener en cuenta: src/app para componentes, src/assets para estáticos, src/environments para configuraciones.

Variables de entorno: No obligatorias, pero puedes usarlas para endpoints u otras configuraciones.

¿Necesita base de datos? No directamente, Angular consume APIs externas.

3

Primeros pasos

“Hola Mundo”: Crea un proyecto con ng new mi-primera-app, ejecuta ng serve --open y edita app.component.html para mostrar “¡Hola Mundo desde Angular!”

4

Utilización del Framework

¿Cómo crear una aplicación base? Usa componentes generados con CLI, define rutas, servicios y vistas.

¿Cómo visualizar resultados? Con ng serve en http://localhost:4200, con recarga en vivo. Para producción, ng build --configuration production.

5

Explicación del funcionamiento

¿Actúa sobre páginas, ficheros, secciones…? Angular usa componentes para secciones de la interfaz, módulos para agruparlos, y enrutamiento para simular páginas sin recargas completas.

¿Cómo se editan los ficheros? Cada componente tiene un TS (lógica), HTML (vista) y CSS (estilo). Edita según requieras, recarga en vivo.

¿Referenciar ficheros desde otras apps? Normalmente Angular se sirve como estáticos finales. Otras apps pueden consumirlo vía iframes o integrarlo en su entorno.

6

Conclusiones

Has aprendido la justificación de Angular, su instalación, primeros pasos, uso, funcionamiento interno y entendido el alcance del tutorial. Con estas bases, puedes comenzar a crear apps complejas con Angular.

7

Enrutamiento

Aprende a configurar rutas en app-routing.module.ts y navegar entre secciones sin recargar la página.

8

Consumo de APIs

Utiliza HttpClient para hacer peticiones a servicios externos y mostrar datos dinámicos en la interfaz.

9

Módulos y Lazy Loading

Divide tu app en módulos y usa lazy loading para mejorar el rendimiento cargando módulos bajo demanda.

10

Directivas y Pipes

Las directivas manipulan el DOM y los pipes transforman datos en la vista. Crea tus propias directivas y pipes personalizados.

11

Pruebas (Testing)

Aprende a escribir pruebas unitarias con Jasmine y Karma, asegurando la calidad del código. Usa ng test para ejecutarlas.

12

Despliegue

Compila con ng build --configuration production y despliega los archivos generados en un servidor estático.

13

Seguridad y Autenticación

Implementa autenticación con JWT, protege rutas con guardas y evita vulnerabilidades XSS mediante binding seguro.

14

SEO y Metadatos

Usa Angular Universal para renderizado del lado del servidor y mejora el SEO. Añade metadatos para optimizar la visibilidad.

15

Internacionalización

Traduce tu app a múltiples idiomas con @angular/localize, generando builds para cada locale.

16

Rendimiento y Optimización

Mejora la performance con lazy loading, OnPush Change Detection y evaluando métricas con Lighthouse.

17

Accesibilidad

Haz tu app usable para todos. Usa roles ARIA, texto alternativo, y asegúrate de un buen contraste y navegación por teclado.

18

Gestión de Estado

Usa NgRx u otras librerías para gestionar el estado global de la app, facilitando la escalabilidad y mantenibilidad.

19

Animaciones

Crea transiciones y efectos visuales con @angular/animations para mejorar la experiencia de usuario.

20

Herramientas del Ecosistema

Aprovecha Angular CLI, Angular Language Service, bibliotecas de componentes (Material, PrimeNG) y más para acelerar el desarrollo.
